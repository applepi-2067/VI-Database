FirmwareInstallation  
Updated Jan 5, 2015 by scottlib...@gmail.com

Requirements

  • navX MXP Circuit Board (rev. 3.3)
  • PC with USB 2.0 port running Windows 7 or greater.
  • Mini-USB Cable

Preparation

  • Download the DFUSE Utility
  • Install the DFUSE Demo using the setup program. Note that this will include installation of a USB driver for communicating to the navX MXP Circuit board, as well as an application. The application is typically installed and on the Windows Start menu at:
Start->All Programs->STMicroelectronics->DfuSe->STDFU Tester

Installing the Firmware

  • Download the latest build of the navx mxp firmware. Extract the files from the zip archive using an unzip program. The navx mxp firmware file is located at:

stm32\bin\navx-mxp_X.Y.ZZZ.hex
X = Major Version Number Y = Minor Version Number Z = Revision NUmber
  1. Connect a USB-mini cable from a Windows PC to the navX MXP circuit board.
    • Verify the red LED labeled “3.3V” comes on
  2. Remove power from the navX MXP (e.g., by disconnecting the USB cable), press and hold down the “CAL” button, then re-apply power. Do not un-press the “CAL” button until a few seconds after the power has been applied.
  3. Run the ST Microelectronics utility “ST DFUTester”
    • Verify the device (“STM Device in DFU Mode”) shows up in the “Devices” list
    • Erase the flash memory
      • Click “Create from Map”
      • Select “Erase” from the Operation List
      • Press Go
      • Verify that the progress bar proceeds from addresses 0x08000000 through 0x0800C000, and then says “Success”
    • Download the firmware to the navX MXP
      • Click “Create from File”
      • Select the navx-mxp_X.Y.ZZZ.hex file; use “Files of Type” Intel HEX File
      • Select “Download” from the Operation List
      • Press Go
      • Verify that the progress bar proceeds from addresses 0x08000000 through 0x0800C000, and then says “Success”
